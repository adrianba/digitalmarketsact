# Digital Markets Act

I am building a website that will allow easy access to the text of the Digital Markets Act (DMA). The DMA is a European Regulation. European Regulations have a list of numbered recitals, followed by a set of Articles. Articles usually have numbered or lettered subparts. Usually each Article or subpart has one or more Recitals associated with it.

The site will have one page per Article. Next to the Article or subparts will be references to the related recitals.

There should be an index page of all the Articles.

The site needs to be clean and professional using easy to read type. The focus should be on the text.

Articles do not have titles, but Articles are grouped under headings. The index page should have the heading followed by its Articles, and then the next heading and its articles and so on.

The focus is on easy navigation between articles (and linking to recitals).

The site will be statically generated using eleventy. Tailwindcss will be used for styling. The tailwind css will be locally generated for the site. Images should favor SVG.

The design files for the web site are in design.zip. However, the design needs tweaking. For example, its title is The Digital Jurist, but we just want the site to be called Digital Markets Act.

The text of the DMA is here: https://eur-lex.europa.eu/legal-content/EN/TXT/HTML/?uri=CELEX:32022R1925
